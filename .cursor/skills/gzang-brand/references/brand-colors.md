## 1. 色彩哲学

### 1.1 归藏青 (Guizang Qing)

**色值**：`#0F4C5C`

归藏青源自中国传统色彩"绀青"，是一种深沉内敛的蓝绿色。它代表着：

- **稳重**：财务管理系统需要传达可信赖感
- **专业**：B2B 企业场景的视觉需求
- **内敛**：不张扬但有力量，符合财务管理的气质

**色彩心理学**：
- 蓝绿色调传递平静、稳定、安全感
- 与水/财富的隐喻相呼应
- 在长时间使用时不会造成视觉疲劳

### 1.2 琉璃金 (Liuli Gold)

**色值**：`#FB8B24`

琉璃金是一种温暖而明亮的橙金色，它代表着：

- **价值**：财富、收益、成功
- **活力**：行动的召唤、积极的信号
- **焦点**：吸引注意力，标记重要信息

**色彩心理学**：
- 暖色调传递温暖、亲切感
- 与"金"的概念直接关联
- 在中性色背景上极具视觉冲击力

### 1.3 利润绿 (Profit Green)

**色值**：`#06D6A0`

一种明亮清新的绿色，代表着：

- **正向**：收入、盈利、增长
- **成功**：交易完成、状态正常
- **希望**：积极向上的财务状况

### 1.4 支出红 (Expense Red)

**色值**：`#EF476F`

一种略带粉调的红色，代表着：

- **负向**：支出、成本、亏损
- **警示**：需要注意的信息
- **紧迫**：需要处理的事项

## 3. 功能色系统

功能色用于传达状态、语义和交互反馈。

### 3.1 状态色

```
成功/利润绿：  #06D6A0  ████████████████  (收入、盈利、成功)
警告/警示黄：  #FFD166  ████████████████  (警告、待处理)
错误/支出红：  #EF476F  ████████████████  (支出、错误、危险)
信息/天蓝：    #118AB2  ████████████████  (信息、提示、链接)
```

### 3.2 语义使用规则

| 语义 | 颜色 | 使用场景 |
|------|------|---------|
| 收入 | `#06D6A0` | 收入金额、成功提示、正向指标 |
| 支出 | `#EF476F` | 支出金额、错误提示、负向指标 |
| 转账 | `#0F4C5C` | 转账金额中性状态 |
| 警告 | `#FFD166` | 警告提示、待审核项 |
| 信息 | `#118AB2` | 信息提示、帮助文档、链接 |

### 3.3 功能色亮度调整

当功能色作为背景时，需要调整不透明度：

```css
/* 浅色背景 */
.bg-success-light {
  background-color: rgba(6, 214, 160, 0.1); /* 10% 透明度 */
}

.bg-danger-light {
  background-color: rgba(239, 71, 111, 0.1);
}

/* 图标容器背景 */
.icon-success {
  background-color: rgba(6, 214, 160, 0.15);
  color: #06D6A0;
}

.icon-danger {
  background-color: rgba(239, 71, 111, 0.15);
  color: #EF476F;
}
```

## 4. 中性色系统

中性色用于文本、背景和边框。

### 4.1 文本色

| 层级 | 色值 | 对比率 | CSS 变量 | 用途 |
|------|------|--------|----------|------|
| 主文本 | `#1F2937` | 15.9:1 | `--gzang-text-primary` | 标题、重要内容 |
| 次文本 | `#6B7280` | 5.74:1 | `--gzang-text-secondary` | 说明文字、标签 |
| 禁用文本 | `#9CA3AF` | 2.58:1 ❌ | — | 仅用于禁用状态 |

**重要**：禁用文本对比度不足 4.5:1，请勿用于正常可读文本。

### 4.2 背景色

| 用途 | 色值 | CSS 变量 | 说明 |
|------|------|----------|------|
| 页面背景 | `#F8F9FA` | `--gzang-bg` | 整体页面背景 |
| 卡片背景 | `#FFFFFF` | `--gzang-surface` | 卡片、面板背景 |
| 深色背景 | `#111827` | — | 深色主题背景 |

### 4.3 边框色

| 色值 | CSS 变量 | 用途 |
|------|----------|------|
| 默认边框 | `#E5E7EB` | 常规边框、分割线 |
| 深色边框 | `#374151` | 深色主题边框 |

## 5. 深色主题

### 5.1 深色主题色板

```css
[data-theme="dark"] {
  /* 品牌色反转 */
  --gzang-primary: #0a3644;      /* 深青 - 主色 */
  --gzang-primary-light: #0F4C5C; /* 归藏青 - 浅色 */
  
  /* 背景色反转 */
  --gzang-bg: #111827;           /* 深灰蓝背景 */
  --gzang-surface: #1F2937;      /* 卡片背景 */
  
  /* 文本色反转 */
  --gzang-text-primary: #F9FAFB; /* 主文本 - 亮色 */
  --gzang-text-secondary: #9CA3AF;/* 次文本 */
  
  /* 边框反转 */
  --gzang-border: #374151;
}
```

### 5.2 深色主题设计原则

1. **避免纯黑**：使用 `#111827` 而非 `#000000`，减少眼睛疲劳
2. **保留语义色**：功能色在深色主题下保持不变
3. **降低对比度**：确保文本对比度仍然 ≥ 4.5:1
4. **减少发光**：避免在大面积深色背景上使用亮色阴影

## 6. 色彩搭配指南

### 6.1 正确搭配

```
✅ 归藏青 + 琉璃金
   - 最佳拍档，专业且有活力
   - 适用于：企业端主色调搭配

✅ 利润绿 + 浅灰背景
   - 清晰醒目，易于识别
   - 适用于：财务指标卡片

✅ 支出红 + 深灰背景
   - 警告效果强，但不刺眼
   - 适用于：错误提示框
```

### 6.2 错误搭配

```
❌ 支出红 + 警示黄
   - 视觉冲突，两色竞争注意力
   - 解决：改用支出红 + 白色背景

❌ 次文本灰 + 浅灰背景
   - 对比度不足，无法阅读
   - 解决：使用 #6B7280 + #FFFFFF

❌ 琉璃金 + 浅色背景
   - 琉璃金需要深色背景衬托
   - 解决：添加深色背景或使用归藏青
```

### 6.3 配色公式

| 场景 | 公式 | 示例 |
|------|------|------|
| 企业端主色 | 归藏青(60%) + 琉璃金(10%) + 中性色(30%) | 侧边栏、标题、CTA |
| 个人端主色 | 琉璃金(40%) + 归藏青(30%) + 中性色(30%) | 主色调、强调元素 |
| 数据展示 | 功能色(30%) + 归藏青(30%) + 中性色(40%) | 图表、指标卡片 |
| 表单设计 | 中性色(70%) + 归藏青(20%) + 功能色(10%) | 输入框、按钮 |

## 7. 应用场景

### 7.1 UI 设计

```vue
<!-- 企业端仪表盘 -->
<div class="bg-background min-h-screen">
  <aside class="bg-primary text-white">...</aside>
  <main>
    <h1 class="text-text-primary">财务概览</h1>
    <div class="bg-surface rounded-xl p-6 shadow-sm">
      <span class="text-success font-mono">+¥12,580.00</span>
      <span class="text-danger font-mono">-¥8,320.00</span>
    </div>
  </main>
</div>
```

### 7.2 PPT / 演示文稿

- **封面背景**：归藏青 `#0F4C5C`
- **标题文字**：白色 `#FFFFFF`
- **强调元素**：琉璃金 `#FB8B24`
- **内容背景**：白色 `#FFFFFF`
- **图表配色**：功能色系统

### 7.3 文档 / 报告

- **标题**：归藏青
- **正文**：深灰 `#1F2937`
- **关键数据**：琉璃金高亮
- **金额**：利润绿(收入)/支出红(支出)

### 7.4 营销物料

- **主视觉**：归藏青渐变背景
- **品牌 logo**：归藏青 + 琉璃金
- **图标**：线性图标，归藏青描边
- **插图**：扁平风格，使用品牌色系
