# G-Zang (归藏) 用户手册 - 快速开始指南

## 欢迎使用G-Zang

G-Zang (归藏) 是一款专为个人和企业用户设计的智能财务管理系统。通过本指南，您将快速了解如何开始使用G-Zang进行记账和财务管理。

---

## 1. 账号注册与登录

### 1.1 注册新账号

1. **访问注册页面**
   - 打开浏览器，访问 G-Zang 官网
   - 点击页面右上角的"注册"按钮

2. **填写注册信息**
   ```
   用户名：输入您的手机号或邮箱地址
   密码：设置8-20位包含字母、数字、特殊字符的密码
   确认密码：再次输入密码
   用户昵称：设置您的显示名称
   ```

3. **验证账号**
   - 如果使用手机号注册：输入收到的短信验证码
   - 如果使用邮箱注册：点击邮箱中的验证链接

4. **完成注册**
   - 验证通过后，系统自动登录
   - 建议完善个人资料（头像、性别、生日等）

### 1.2 登录账号

1. **访问登录页面**
   - 在官网点击"登录"按钮

2. **输入登录信息**
   ```
   用户名：注册时使用的手机号或邮箱
   密码：您的登录密码
   ```

3. **安全登录**
   - 如启用了"记住我"，可选择保存登录状态
   - 首次登录新设备可能需要短信验证

## 2. 个人记账快速入门

### 2.1 添加第一个账户

1. **进入账户管理**
   - 登录后，点击左侧菜单"账户管理"
   - 点击"添加账户"按钮

2. **填写账户信息**
   ```
   账户名称：如"现金"、"招商银行卡"
   账户类型：选择现金、银行卡或电子支付
   初始余额：输入账户当前余额（可选）
   ```

3. **保存账户**
   - 点击"保存"按钮完成添加
   - 可以添加多个账户进行资金管理

### 2.2 记录第一笔交易

1. **快速记账入口**
   - 在首页点击"+"按钮或"记账"标签
   - 选择"收入"或"支出"

2. **填写交易信息**
   ```
   金额：输入交易金额（如：25.50）
   分类：选择交易分类（如餐饮、购物）
   账户：选择资金账户
   交易时间：选择发生时间（默认为当前时间）
   备注：添加交易说明（可选）
   ```

3. **保存交易**
   - 点击"保存"按钮完成记账
   - 系统自动更新账户余额

### 2.3 查看收支统计

1. **访问统计页面**
   - 点击底部导航栏"统计"标签

2. **查看各类报表**
   - **收支概览**：本月收入、支出和结余
   - **分类统计**：按类别查看消费分布
   - **趋势图表**：查看收支变化趋势
   - **账户余额**：各账户余额汇总

## 3. 企业用户入门

### 3.1 创建企业

1. **申请企业版**
   - 在个人中心点击"创建企业"
   - 填写企业基本信息：
     ```
     企业名称：您的公司名称
     行业类型：选择所属行业
     企业规模：选择员工数量范围
     ```

2. **完善企业信息**
   - 上传企业营业执照（可选）
   - 设置企业Logo和介绍

### 3.2 邀请团队成员

1. **进入成员管理**
   - 在企业管理页面点击"成员管理"

2. **邀请新成员**
   - 点击"邀请成员"按钮
   - 输入成员邮箱或手机号
   - 选择成员角色（管理员、财务、普通成员）

3. **成员加入流程**
   - 系统发送邀请链接给成员
   - 成员点击链接注册账号
   - 您审核通过后成员正式加入

### 3.3 企业记账

1. **关联业务记账**
   - 在记账时选择"关联业务"
   - 输入工单号或业务编号
   - 系统自动关联业务和财务数据

2. **成本中心分配**
   - 选择支出所属的成本中心
   - 支持多级成本中心结构

## 4. 高级功能使用

### 4.1 预算管理

1. **设置预算**
   - 进入"预算管理"页面
   - 点击"新增预算"
   - 选择预算类型：
     - **月度预算**：每月消费上限
     - **分类预算**：特定类别预算
     - **年度预算**：全年消费规划

2. **预算监控**
   - 系统自动计算预算使用率
   - 达到80%时发送提醒通知
   - 超预算时可设置是否允许继续消费

### 4.2 数据导出

1. **导出交易记录**
   - 在交易记录页面点击"导出"
   - 选择导出格式：Excel 或 PDF
   - 选择时间范围和数据类型

2. **生成财务报告**
   - 在统计页面点击"导出报表"
   - 选择报告类型：月度报告、年度报告
   - 系统自动生成专业财务报表

## 5. 移动端使用指南

### 5.1 APP下载安装

1. **下载方式**
   - **iOS用户**：App Store搜索"G-Zang"
   - **Android用户**：各大应用市场搜索"G-Zang"或扫描二维码下载

2. **首次使用**
   - 安装完成后打开APP
   - 使用已注册账号登录
   - 开启通知权限（用于预算提醒）

### 5.2 移动端特色功能

1. **拍照识单**
   - 点击"记账" → "拍照识单"
   - 对准小票拍照
   - 系统自动识别金额和商品信息

2. **语音记账**
   - 长按"记账"按钮
   - 说出记账内容，如"吃饭花了25块钱"
   - 系统自动解析并生成交易记录

3. **离线记账**
   - 无网络时仍可记账
   - 数据自动同步到云端
   - 支持断网续传

## 6. 常见问题解决

### 6.1 登录问题

**忘记密码**
1. 在登录页面点击"忘记密码"
2. 输入注册邮箱或手机号
3. 按提示重置密码

**账号被锁定**
- 密码错误次数过多导致锁定
- 等待30分钟自动解锁，或联系客服

### 6.2 记账问题

**交易记录错误**
1. 找到错误记录点击编辑
2. 修改错误信息
3. 保存后系统自动更新统计

**分类找不到**
1. 进入"设置" → "分类管理"
2. 点击"添加分类"
3. 创建自定义分类

### 6.3 数据同步问题

**数据不同步**
1. 检查网络连接
2. 在设置中点击"同步数据"
3. 如问题持续，请联系客服

## 7. 安全使用建议

### 7.1 账号安全
- 🔒 **定期修改密码**：建议每3个月修改一次
- 📱 **开启两步验证**：在设置中启用短信验证
- 🚫 **不要在公共设备保存密码**
- 📧 **及时查看安全通知**

### 7.2 数据安全
- 💾 **定期备份数据**：导出重要财务数据
- 🔐 **保护隐私信息**：不要随意分享财务详情
- 📱 **谨慎授权**：只给可信任的应用授权
- 🗑️ **及时清理**：删除不需要的记录和文件

### 7.3 使用习惯
- ✅ **每日记账**：养成良好记账习惯
- 📊 **定期查看报表**：了解财务状况
- 🎯 **合理预算**：控制消费支出
- 📈 **分析趋势**：发现消费规律
