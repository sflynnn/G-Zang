## 1. 审查检查清单

### 🔴 必须通过（正确性 + 安全性）

- [ ] 代码实现了需求文档描述的功能
- [ ] 边界条件被正确处理（空值、0、最大值、负数）
- [ ] 所有 SQL 使用参数化查询
- [ ] 无硬编码密码/密钥/Token
- [ ] 权限检查在正确位置
- [ ] 错误信息不泄露内部实现
- [ ] 事务边界正确（避免部分成功部分失败）

### 🟡 建议检查（性能 + 可维护性）

- [ ] 无 N+1 查询问题
- [ ] 无全表扫描的查询
- [ ] 未使用 SELECT *
- [ ] 分页使用游标而非 OFFSET
- [ ] 无重复代码应提取为公共方法
- [ ] 无魔法数字应定义为常量
- [ ] 变量/函数命名清晰表达意图
